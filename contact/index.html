Frequently Asked Questions

Find answers to common questions about Velyzo

What is Velyzo?

Velyzo is a coding organization founded by Devin Oldenburg, focused on developing mobile applications and tools. We create practical software solutions for iOS, web platforms, and developer tools.

What projects does Velyzo work on?

Our current projects include Connecto (HTTP request tool for iOS), Velink (Linktree alternative), Stryd (step counter app), and PiVision (Raspberry Pi monitoring for iOS). All projects are available on our GitHub.

Are your projects open source?

Yes! Our projects are open source and available on GitHub at github.com/velyzo. We believe in contributing to the developer community and encouraging collaboration.

What technologies do you use?

We primarily use Swift for iOS development, React and Next.js for web applications, and React Native for cross-platform mobile development. We select the most appropriate technology for each project's requirements.

Can I contribute to your projects?

Absolutely! We welcome contributions from developers. Check out our GitHub repositories, read any contribution guidelines, and feel free to submit issues, feature requests, or pull requests.

How can I stay updated with your projects?

Follow us on X/Twitter (@velyzo_official), join our Discord community, and star our GitHub repositories to get notified of new releases and updates.

How do I report bugs or request features?

Please use the GitHub Issues section of the relevant project repository. This helps us track and prioritize issues effectively. You can also discuss ideas in our Discord server.

What licenses do your projects use?

Please check the specific license for each project on GitHub. Most projects use permissive open-source licenses, but it's important to review the terms for each individual project.
